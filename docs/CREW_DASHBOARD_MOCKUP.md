## Color Scheme

The dashboard uses a clean, professional color scheme:

- **Primary Blue**: `#1f77b4` - Headers, main accents
- **Teal**: `#2c7fb8` - Check-in graphs
- **Orange**: `#ff7f50` - Day pass trends
- **Yellow**: `#ffc107` - Warning indicators
- **Green**: `#2ca02c` - Success indicators
- **Red**: `#d62728` - Critical alerts

**Backgrounds**:
- White cards with subtle shadows
- Light gray `#f0f2f6` for metric cards
- Yellow `#fff3cd` for at-risk warnings

## Responsive Design

The dashboard is fully responsive:

- **Desktop**: Full width, 2-column layouts
- **Tablet**: Stacked sections, 1-2 columns
- **Mobile**: Single column, optimized for vertical scrolling
