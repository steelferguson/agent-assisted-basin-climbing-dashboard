# Basin Climbing Data Pipeline

Data pipeline for Basin Climbing & Fitness. Fetches data from 13+ external sources and uploads to S3.

## Data Sources

- **Capitan** - Gym management (memberships, check-ins, members)
- **Stripe** - Payment processing
- **Square** - POS transactions
- **Shopify** - E-commerce orders
- **Mailchimp** - Email marketing
- **Klaviyo** - Marketing automation
- **SendGrid** - Transactional email
- **Twilio** - SMS messaging
- **Google Analytics 4** - Website analytics
- **Instagram** - Social media
- **Facebook Ads** - Ad performance
- **QuickBooks** - Accounting
- **Firebase** - Birthday party RSVPs
